# Bybit 레버리지 선물 자동매매 봇

기술적 분석과 거시경제 분석을 결합한 ETH 선물 자동매매 프로그램입니다.

---

## 전략 개요

```
30초마다 반복
  ↓
[1] 기술적 분석 (1시간봉)         → LONG / SHORT / NEUTRAL
[2] 추세 필터  (4시간봉 EMA50)    → LONG / SHORT
[3] 거시경제 분석 (10분 캐싱)
    - Fear & Greed Index
    - CoinDesk / CoinTelegraph RSS 뉴스 (최신 5개)
    - Groq AI (Llama 3.3 70B) 종합 판단 → LONG / SHORT / NEUTRAL
    ※ 토큰 절약을 위해 10분에 1번 호출, 나머지는 캐싱 결과 재사용
  ↓
세 가지가 모두 같은 방향 → 진입
반대 신호 발생 → 청산
Hard SL 발동 (-3.0%) → 즉시 강제 청산
```

## 기술 지표 (점수 기반, 최대 ±10점)

| 지표 | 가중치 | LONG 조건 | SHORT 조건 |
|------|--------|-----------|------------|
| EMA 9/21/50 배열 | ±2 | 정배열 + 가격 > EMA50 | 역배열 + 가격 < EMA50 |
| RSI(14) | ±2 | 35 이하(과매도) | 75 이상(과매수) |
| MACD 히스토그램 | ±1 | 양수 + 상승 확대 | 음수 + 하락 확대 |
| 볼린저밴드 위치 | ±2 | 하단 터치(+2) / 50% 이하(+1) | 상단 돌파(-2) / 50% 이상(-1) |
| 일목균형표 구름대 | ±2 | 구름대 위 | 구름대 아래 |
| 거래량 급증 (1.5x↑) | ±1 | 상승봉 + 급증 | 하락봉 + 급증 |

> **5점 이상 → LONG / -5점 이하 → SHORT / 그 사이 → NEUTRAL**

## 진입 조건 (전부 충족해야 진입)

| 조건 | LONG | SHORT |
|------|------|-------|
| 1시간봉 신호 | LONG (RSI 35 이하 + BB 50% 이하 포함) | SHORT (RSI 75 이상 + BB 50% 이상 포함) |
| 4시간봉 트렌드 | EMA50 위 | EMA50 아래 |
| Groq AI 판단 | LONG 또는 NEUTRAL | SHORT 또는 NEUTRAL |
| Fear & Greed | 조건 없음 | 조건 없음 |

## 분석 데이터 소스 및 기준

### 기술적 분석 — Bybit API (직접 계산)

| 데이터 | 출처 | 판단 기준 |
|--------|------|----------|
| ETH 1시간봉 캔들 | Bybit API | EMA/RSI/MACD/볼린저/일목균형표/거래량 계산 |
| ETH 4시간봉 캔들 | Bybit API | EMA50 위/아래로 큰 추세 판단 |

### 거시경제 분석 — 외부 데이터

| 데이터 | 출처 사이트 | 판단 기준 |
|--------|------------|----------|
| Fear & Greed Index | alternative.me | 80↑ 극단 탐욕 → 숏 경고 / 20↓ 극단 공포 → 롱 기회 |
| ETH 뉴스 헤드라인 | CoinDesk RSS, CoinTelegraph RSS | 최신 5개 자동 수집, API 키 불필요 |
| AI 종합 판단 | Groq API (Llama 3.3 70B) | 뉴스 + Fear&Greed 기반으로 아래 5가지 분석 |

### Groq AI 분석 기준 5가지

1. **연준(Fed) 정책** — 금리 인상/동결/인하 기대감
2. **달러 강세(DXY)** — 달러 강세면 ETH 약세 압력
3. **기관 투자자 동향** — ETF 자금 유출입, 대형 매수/매도 신호
4. **규제/지정학 리스크** — 각국 정부 정책, 글로벌 이벤트
5. **시장 심리** — Fear & Greed 수치 해석

→ 최종적으로 `LONG / SHORT / NEUTRAL` + 신뢰도(0~100%) 반환

## 설정 방법

프로젝트 폴더에 `.env` 파일을 생성하고 아래 내용을 입력합니다.

```
BYBIT_API_KEY=발급받은키
BYBIT_SECRET_KEY=발급받은시크릿
GROQ_API_KEY=발급받은키
TELEGRAM_TOKEN=텔레그램봇토큰
TELEGRAM_CHAT_ID=텔레그램챗ID
```

> `.env` 파일은 `.gitignore`에 포함되어 GitHub에 올라가지 않습니다.

그 외 설정은 `bybit_autotrading.py` 상단에서 수정합니다.

```python
SYMBOLS         = ["ETHUSDT"]  # 거래 심볼
LEVERAGE        = 5            # 고정 레버리지
SL_PCT          = 0.020        # 거래소 스탑로스 비율 (2.0%)
HARD_SL_PCT     = 0.030        # 봇 루프 Hard SL 비율 (3.0%)
TP_RATIO        = 1.5          # TP 비율 (SL_PCT × TP_RATIO = 3.0%)
NEWS_COUNT      = 5            # Groq에 전달할 뉴스 개수
LOOP_SEC        = 30           # 봇 반복 주기 (초)
MACRO_CACHE_SEC = 600          # 거시경제 분석 캐싱 주기 (초, 10분)
TESTNET         = False        # True = 테스트넷 사용
DRY_RUN         = False        # True = 드라이런 (실제 주문 없음)
```

## Claude → Groq 전환 이유

초기에는 Claude API(Anthropic)를 사용했으나 아래 이유로 Groq으로 전환했습니다.

- **Claude API**: 무료 플랜 없음, 크레딧 소진 후 유료
- **Gemini API**: 한국 리전에서 무료 티어 할당량 0으로 제한됨
- **Groq API**: 한국 포함 전 세계 무료 제공, 응답 속도 빠름

Groq은 자체 LPU(Language Processing Unit) 칩 기반으로 오픈소스 모델(Llama 3.3 70B)을 무료로 호스팅합니다.
